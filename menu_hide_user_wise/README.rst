Hide Menu User
==============
The Hide Menu User module for Odoo 19 lets administrators hide selected menu
and submenu items for specific users. It helps keep the backend simpler for
users who should not see specific navigation entries.

Installation
============
- www.odoo.com/documentation/19.0/setup/install.html
- Install the module in your custom addons path

Configuration
=============
- No additional configuration is required
