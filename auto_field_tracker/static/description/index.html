How It Works

Live in 4 steps — no restart, no code

From installation to your first tracked change in under 10 minutes.

1

Install the module

Add auto_field_tracker to your addons path and install from the Apps menu. Odoo 17, 18, and 19 on both Community and Enterprise. Only dependencies are base and mail.

2

Create a tracker config

Go to Settings → Technical → Auto Field Tracker and click New. Select the model you want to monitor. Only chatter-enabled models appear in the list — no invalid configs possible.

💡 Create as many configs as you need — one per model, all fully independent.
3

Choose fields and events

Enable Track ALL Fields for full coverage, or use the field picker to select only what matters. Toggle which events — creation, updates, deletions — are logged. Optionally enable One2many line tracking and follower notifications.

4

Save — tracking starts immediately

Hit Save and the tracker goes live. Use the built-in Test Tracking button to confirm a test entry appears in the Chatter. No Odoo restart at any point in this process.

💡 Use View Logs on the config form to open a pre-filtered audit log scoped to that specific model.
