All in login page custom
==========================

Customize the Odoo 17 login page with configurable login box positions and
background styles. Administrators can choose the default Odoo login page or
display the login form on the left, middle, or right side of the screen.

Features
========

* Select Default, Left, Middle, or Right login form orientation.
* Use a solid background color from the settings color picker.
* Upload a background image from the system.
* Use an external image URL as the login page background.
* Preserve Odoo's standard login, database selector, messages, and footer flow.

Configuration
=============

1. Install the module in Odoo 17.
2. Go to Settings.
3. Open the Login Page Set-Up section.
4. Select the login box orientation.
5. Choose a background type when using a custom orientation.
6. Save the settings and open ``/web/login`` to review the result.
